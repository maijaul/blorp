Margin

Margin er luft rundt innholded. Denne luften er usynlig.

Border

Border The CSS border properties allow you to specify the style, width, and color of an element's border. The border property is a shorthand property for:border-width border-style (required) border-color If border-color is omitted, the color applied will be the color of the text.

Boksmodellen

Boksmodellen i CSS er en boks rundt hver HTML element. Den inneholder margins, borders, padding, and the actual content. Bildet under illustrerer boksmodellen:

Selector

Selector er kodene man skriver inn for å velge hvilke elementer man ønsker å designe eller spesifisere

Attribute

CSS Attribute selector brukes til å velge et element med en spesifikk egenskap eller for å tillegge en verdi. Det er en god måte å style HTML-elementer ved å gruppere de basert på spesifikke attributes

Value

Values er en del av Css reglene til CSS. Tillater bruk av tall, lengder, prosent, URLs, “Counters”, og farger.

p taggen

P - står for paragraph og brukes oftest til mengdeteksten. Her kan man skrive generell tekst.

Hva brukes flexbox i css til?

Flexbox gjør det lettere å designe layout på nettsiden. Mer effektiv måte å plassere “Align” osv. Funker bra selv om størrelser er ukjente og/eller dynamisk.
